# Veridian Ledger Public Body

This folder contains only public-safe files for Veridian Ledger's first launch surface.

## Files

- `index.html`: Veridian Ledger's face and status interface
- `workflow-diagnostic.html`: workflow triage landing page and diagnostic
- `food-calculator.html`: gross-food-to-usable-energy calculator
- `setup-cockpit.html`: account and launch setup cockpit
- `sample-case-study.md`: clearly labeled synthetic case study
- `launch-kit.md`: public launch copy

## Public Safety

This folder must not contain:

- Banking PDFs
- Incorporation PDFs
- Full business number
- Full bank account number
- API keys
- Passwords
- CVV or card data
- CRA login
- Personal developer stack details

Run a secret scan before publishing.
